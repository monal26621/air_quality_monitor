# air_quality_monitor
The Air Quality Monitoring System is a Machine Learning-based web application that predicts air quality levels using environmental parameters. The project helps users analyze pollution conditions and understand whether the air quality is safe or harmful.

This project is built using:

Python
Machine Learning
Flask Framework
HTML/CSS
Scikit-learn

The application takes user inputs through a web interface and predicts air quality using a trained ML model.

 **Features**
 Predicts air quality based on user inputs
Interactive web interface using Flask
Machine Learning model integration
Data preprocessing with scaler
Easy deployment structure
Beginner-friendly project structure
